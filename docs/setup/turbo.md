## Installing Turbo + Stimulus

This project uses Hotwire (Turbo + Stimulus) to build a fast and interactive UI without a full SPA framework.

## Frontend setup (Webpack Encore + Stimulus)

This project uses Webpack Encore to build frontend assets, including Stimulus controllers and Turbo integration.  
Although Stimulus runs in the browser, it is bundled and compiled through the backend (Symfony) build system.  

### Installation
`npm install`  
`npm install --save-dev @symfony/webpack-encore`  

- Project launch  
`npm run dev`
